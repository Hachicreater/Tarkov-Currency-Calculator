# タルコフ通貨計算機

Escape from Tarkov のゲーム内通貨 **RUB / USD / EUR** を相互に換算しつつ、  
トレーダーの買取価格からレートを算出する Web ツールです。  
tarkov.dev の GraphQL API を使用した **非公式のツール** です。

---

## 🔧 主な機能

- **通貨換算**
  - RUB / USD / EUR のいずれかを入力すると、残り 2 通貨を自動計算
  - 基準レートは Tarkov のトレーダー買取価格（Peacekeeper=USD, Skier=EUR）

- **レートの手動調整**
  - `config/rates-config.json` でオフセット・固定レートを設定可能
  - 画面上でも「1 USD = RUB」「1 EUR = RUB」を手動入力して即反映
    ※APIで取得した金額とゲーム内の金額が異なるため、この機能を追加しております

- **アイテム買取価格の検索**
  - アイテム名（英語）で検索し、**トレーダーのみ**の買取価格を表示
  - ユーザーが販売しているMarketの価格は無視
  - 最も高く買い取るトレーダー 1人だけを表示
  - トレーダーのアイコン画像・名前も表示
  - 入力中に候補サジェストを表示（英語表記）
